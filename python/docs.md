## How-to-use

### 시작하기
```python
import chojungjong as cjj
```

### 분리

#### sep(letter: string)
```python
cjj.sep('와')
```

#### sfa(word: string)
```python
cjj.sfa('안녕하세요')
```

### 결합

#### con(frag: list)
```python
cjj.con(['ㅇ', 'ㅘ'])
```

#### cfa(fragArray: list)
```python
cjj.cfa([['ㅎ', 'ㅏ', 'ㄴ'], ['ㄱ', 'ㅡ', 'ㄹ']])
```

#### cfs(fragString: string)
```python
cjj.cfs('ㅇㅏㄴㄴㅕㅇㅎㅏㅅㅔㅇㅛ')
```
